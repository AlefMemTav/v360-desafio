% Slide 5: Detalhes da Implementação
\begin{frame}{Por que essa Stack?}
    Explorando os diferenciais da vaga para garantir \textbf{\textcolor{v360orange}{performance}}:
    
    \vspace{0.5cm}
    \begin{itemize}
        \setlength\itemsep{0.5em} % Aumenta espaço entre itens
        \item \textbf{Ruby on Rails (API):} Robustez e padrão MVC.
        \item \textbf{Sidekiq + Redis:} Processamento em \textit{background} para não travar a UI (\texttt{HTTP 202}).
        \item \textbf{ActionCable (WebSocket):} Entrega da resposta em tempo real.
        \item \textbf{PostgreSQL:} Segurança e integridade dos dados fiscais.
    \end{itemize}
\end{frame}
